# ai-cli

Simple CLI for ChatGPT.

## Requirements:

- Node version >= `18`

## Usage:

1. Clone this repository.
2. Add your OpenAI API key in an `.env` file. A template is provided.
3. (Optional) Add the script to your path.
4. `ai <message>` (`chat` is already in use on OSX.)
